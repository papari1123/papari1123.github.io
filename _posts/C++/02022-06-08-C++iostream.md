# Introduction
C++에서 입출력 클래스를 정의한 iostream에 대해 알아본다.

# 출력 :cout
삽입 연산자 <<를 활용한다.
```cpp
#include <iostream>
using namespace std;
cout << 출력할데이터 ;
```

# 입력 : cin
추출 연산자 >>를 활용한다.
```cpp
#include <iostream>
using namespace std;
cin >> 입력할데이터;
```
