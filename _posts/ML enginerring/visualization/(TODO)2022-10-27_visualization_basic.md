# Introduction


# 시각화를 진행할 데이터는
- 데이터셋 관점 (global)에서 분석이 가능
- 개별 데이터 셋 관점(local)에서도 분석 가능

# 데이터 종류
- 시계열 데이터 : 추세, 계절성, 주기성 살핌
- 지리/지도 데이터
- 관계 데이터 : 객체는 node, 관계는 link (크기, 색, 수 등으로 관계의 가중치 표현)

# 마크와 채널
- 마크는 점,선,면으로 이루어진 데이터를 시각화한다.
- 채널은 각 마크를 변경할 수 있는 요소이다. (위치, 색, 모양 등)
